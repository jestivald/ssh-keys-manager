# 🔑 SSH Keys Manager

Простой PowerShell-скрипт для управления SSH-ключами нод RemnaWave (и не только) на Windows 11.

Каждый ключ лежит в своей именованной папке, удобно когда нод много.

## ✨ Возможности

- 🔑 Генерация Ed25519 ключей (по одному или пачкой)
- 📂 Автоматическое создание папок `имя-ноды/имя-ноды` + `.pub`
- 📋 Публичный ключ сразу в буфере обмена
- 📑 Список всех созданных ключей с датами
- 🗑️ Удаление ключа по номеру или имени
- 💾 Запоминает путь к папке (спрашивает один раз)
- 🎨 Красивый интерактивный интерфейс

## 📦 Структура файлов

После запуска получится такая структура:

```
ssh-keys/
├── ru-node/
│   ├── ru-node          (приватный ключ)
│   └── ru-node.pub      (публичный ключ)
├── de-node/
│   ├── de-node
│   └── de-node.pub
└── us-node/
    ├── us-node
    └── us-node.pub
```

## 🚀 Установка

### Вариант 1: Скачать напрямую
```powershell
# Скачиваем скрипт в любую удобную папку
irm https://raw.githubusercontent.com/jestivald/ssh-keys-manager/main/ssh-keys.ps1 -OutFile ssh-keys.ps1
```

### Вариант 2: Клонировать репозиторий
```powershell
git clone https://github.com/jestivald/ssh-keys-manager.git
cd ssh-keys-manager
```

### Разрешение на запуск скриптов (если ещё не настроено)
PowerShell по умолчанию блокирует запуск скриптов. Выполни один раз от **админа**:
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

## 🎯 Использование

### Интерактивное меню (проще всего)
```powershell
.\ssh-keys.ps1
```

Получишь меню с выбором действий.

### Прямые команды
```powershell
.\ssh-keys.ps1 new      # Создать ключ(и)
.\ssh-keys.ps1 list     # Список всех ключей
.\ssh-keys.ps1 remove   # Удалить ключ
```

## 💡 Запуск одной командой (алиас в профиле)

Чтобы вызывать скрипт из любой папки одной командой `ssh-keys`, добавь в свой PowerShell-профиль:

```powershell
# Открываем профиль
notepad $PROFILE
```

Вставляем функцию (подставь свой путь к скрипту):
```powershell
function ssh-keys {
    & "C:\путь\к\ssh-keys.ps1" @args
}
```

Перезапускаем терминал, теперь работает:
```powershell
ssh-keys        # меню
ssh-keys new    # создать
ssh-keys list   # список
```

## ⚙️ Требования

- Windows 10/11
- PowerShell 5.1+ (или PowerShell 7+)
- OpenSSH Client (обычно предустановлен; если нет: Settings → Apps → Optional features → OpenSSH Client)

## 🔒 Безопасность

- Используется алгоритм **Ed25519**: современный, быстрый, безопасный
- Ключи создаются **без passphrase** (для автоматизации). Если нужна парольная защита, открой скрипт и измени параметр `-N '""'` на `-N` (или допиши запрос пароля)
- Приватные ключи (файлы без `.pub`) **никогда никому не передавай**
- На сервер вставляется только содержимое файла `.pub`
